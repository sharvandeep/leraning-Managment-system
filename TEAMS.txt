# Team Members Task Division

The LearnSphear Learning Management System project was developed by a team of four members. To ensure efficient project execution, responsibilities were divided based on different phases of software development.

| **Member**               | **Primary Responsibility**                        | **Tasks Performed**                                                                                                                                                                                                                                                                                |
| ------------------------ | ------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Member 1               | **Requirement Analysis & System Workflow Design** | Studied the project problem statement, gathered requirements, designed the overall LMS workflow, defined user roles (Student, Teacher, Admin), prepared the database planning, coordinated the team, finalized feature requirements, managed API planning, and supervised project integration.     |
| **Member 2**             | **Frontend Development**                          | Designed and developed the complete user interface using React.js, implemented responsive pages for Student, Teacher, and Admin portals, integrated frontend components with backend APIs, handled routing, authentication screens, dashboards, and user experience improvements.                  |
| **Member 3**             | **Database Design & Backend Development**         | Designed the PostgreSQL database schema, implemented the FastAPI backend, developed REST APIs, implemented JWT authentication, role-based access control, course management, assignment management, quiz management, AI integration APIs, and database operations.                                 |
| **Member 4**             | **Deployment & System Integration**               | Managed project deployment, configured GitHub repositories, deployed the React frontend and FastAPI backend, configured PostgreSQL database hosting, managed environment variables, performed end-to-end testing, resolved deployment issues, and ensured successful frontend-backend integration. |

### Team Collaboration

Although responsibilities were divided among team members, all members actively collaborated during system design, feature discussions, testing, debugging, documentation, and final project integration to ensure the successful development of the LearnSphear Learning Management System.
