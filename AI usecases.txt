AI Integration Report
LearnSphear – AI-Powered Learning Management System
1. Introduction
Artificial Intelligence has been integrated into the LearnSphear Learning Management System (LMS) to enhance the learning experience, automate academic tasks, and provide intelligent educational assistance for students, teachers, and administrators.
Instead of functioning as a simple chatbot, the AI component understands the current academic data stored in the database and generates meaningful learning assistance, quiz generation, grading suggestions, academic risk analysis, and study material summaries. The integration is built using a Unified AI Configuration Backend, supporting Google Gemini, Ollama (Local LLM), and Simulator Mode, with multiple reliability mechanisms to ensure uninterrupted AI services even during API failures, quota limitations, or internet connectivity issues.
________________________________________
2. AI Architecture
The AI module follows a layered architecture.
React Frontend
        │
       ▼
FastAPI Backend
        │
       ▼
AI Controller
        │
        ├── Academic Context Builder
        ├── Prompt Engineering
        ├── AI Provider Selection
        ├── Unified AI Configuration
        ├── Gemini API
        ├── Ollama (Local LLM)
        └── Simulator Mode
The backend first collects relevant academic data from the PostgreSQL database, such as student performance, course information, assignments, quizzes, and uploaded learning materials. This information is converted into a structured context, which is then provided to the selected AI provider to generate intelligent, context-aware responses for the frontend.
This architecture enables flexible AI provider selection, improves response quality, and allows the system to operate using cloud-based AI, local AI models, or simulator mode based on user preferences.

4.1 AI Study Tutor & Interactive Chat Assistant
Purpose
A floating AI Study Tutor is available throughout the student portal to answer academic questions in natural language and provide personalized learning assistance.
Unlike a traditional chatbot, the AI understands the student's academic context and can generate quizzes, explain concepts, and assist with course-related learning.
How it Works
1.	Student asks an academic question or enters a study topic.
2.	Backend collects:
o	Student profile
o	Course information
o	Uploaded learning materials
3.	Academic context is prepared.
4.	Context is inserted into the AI prompt.
5.	The selected AI provider generates an educational response.
Example
Student:
How do JavaScript Promises work?
AI:
Promises are objects used to handle asynchronous operations in JavaScript...
Benefits
•	Natural language learning assistance
•	Personalized academic support
•	Interactive quiz generation
•	Available throughout the student portal

Instead of sending raw user queries directly to the AI model, the backend prepares structured prompts by combining the user's request with relevant academic context.
Example Prompt
You are an AI Learning Assistant.
Analyze the following academic information.
Student Branch:
...
Course:
...
Study Topic:
...
Provide a clear explanation and learning assistance.
This approach improves the quality of AI-generated responses while providing context-aware educational assistance.
________________________________________
6. AI Provider Selection
Instead of relying on a single AI provider, LearnSphear allows users to select the preferred AI model through the Unified AI Configuration panel.
Available AI Providers
● Google Gemini (Cloud AI)
● Ollama (Local LLM)
● Simulator Mode
This provides flexibility, improved privacy, and uninterrupted AI services based on user preferences.
________________________________________
7. Simulator Mode
If cloud or local AI services are unavailable, the application supports a built-in Simulator Mode.
Simulator Mode generates structured educational responses for:
● Study Assistance
● Quiz Generation
● Material Summarization
● Assignment Feedback
This enables demonstrations and development without requiring an active AI service.

12. Conclusion
The AI integration transforms the LearnSphear Learning Management System from a conventional LMS into an intelligent educational platform. By combining academic data with modern Large Language Models such as Google Gemini and Ollama, the system provides personalized tutoring, intelligent quiz generation, AI-assisted grading, academic risk analysis, and automated study material summarization.
The Unified AI Configuration Backend allows users to switch seamlessly between cloud-based AI, local AI models, and Simulator Mode, providing flexibility and reliable AI assistance. Overall, the AI component enhances teaching efficiency, improves student learning, and demonstrates the practical integration of generative AI into modern educational systems.
